# 🛍️ Bliss & Beauty

A modern beauty products e-commerce platform built with React, Tailwind CSS, Firebase Authentication, and Strapi Headless CMS.

Bliss & Beauty aims to provide a seamless online shopping experience for beauty and skincare enthusiasts through an intuitive interface, product discovery features, personalized recommendations, and secure authentication.

---

## 📌 Project Overview

Bliss & Beauty is a full-stack e-commerce application developed as an internship project to gain hands-on experience with modern web technologies, frontend architecture, backend content management systems, authentication, API integration, and scalable application design.

The project follows a Headless CMS architecture where React handles the frontend user experience and Strapi manages product and content data through REST APIs.

## 🔧 Installation

### Clone Repository

```bash
git clone https://github.com/aiera123/bliss-and-beauty.git
```

### Navigate to Project

```bash
cd bliss-and-beauty
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
