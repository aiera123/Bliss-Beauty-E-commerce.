## 🚀 Features

- Modern responsive UI

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript


---

## ⚙️ Installation



Go to project folder:

```bash
cd ecommerce-app
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
